qa-openstack-ghost
==================

Dependencies
------------

- PyQt or PySide
- Ghost.py (available via pip)

Running
-------

- Clone this repo somewhere
- Set correct `DASHBOARD_URL` in config
- Call `./run.sh`

Ghost.py
--------

Ghost.py is an easy replacement for selenium.
The Documentation can be found here:
http://jeanphix.me/Ghost.py/
